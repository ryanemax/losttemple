各投资品种比较

  1. 银行存款、国债
    • 万元，年亏损300元，人民币通胀率5-6%
  2. 股票、期货
    • 2017后小幅上升
  3. P2P（推荐）
    • 年收益 15% 存在欺骗、非法集资等风险
    • 年收益 10-13% 可信 => 认准平台 => 赶超通涨的收益率
  4. 贵金属、原油
    • 国内 => 无规范 => 无公信力 强烈不推荐
    • 国外 => 认准平台 => 新西兰、英国的监管标准完善
  5. 外汇（推荐）
    • 中国
    • 1994年，灰色地带，无限制无监管
    • 2005年，国外公司大量涌入
    • 2008、2009之后 更多人接触到
    • 2016年未完全开发/行业起步较晚、处于发展阶段
    • 国内金融的最后一块蛋糕
      • 分析师 100万/年收入
      • 交易员 300-500万/年收入
      • IB代理
      • 业务员 200-300万/年收入
