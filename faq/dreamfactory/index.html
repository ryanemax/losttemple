Composer Install 依赖问题

问题分析

  1. php -m | grep “mongo”

正常显示mongo插件

  1. php –ini

正常显示20-mongo.ini插件

解决方案

apt-get install php5-dev

apt-get install libssl-dev

apt-get install pkg-config

pecl install mongodb

vim /etc/php5/cli/php.ini

# add this line
# extension = mongodb.so

composer install
