如何在Bluemix上搭建parse-server服务

参考资料

  • Parse on IBM Bluemix
  • Migrating from Parse to Bluemix
  • Parse server migration to IBM bluemix

自动配置步骤

git clone https://github.com/ibm-cds-labs/parse-on-bluemix.git
cf push Your-App-Name-On-Bluemix

手动配置步骤

  1. 注册Bluemix账号 http://bluemix.net
  2. 创建Cloud Foundry应用
    • 选择Web Application
    • 但不选择任何移动服务（Mobile Client Access等）
  3. 选择SDK for Node.js
    • APP Name 起名 parse-on-bluemix
    • 点击完成/FINISH
  4. 创建MongoDB实例（或直接使用已有数据库）
    • 在parse-on-bluemix中选择，ADD A SERVICE OR API
    • 寻找分类 Data and Analytics
    • 选择新建 MongoDB by Compose
    • 按需配置
    • 本文不详细介绍数据库配置
  5. 配置parse-server环境变量
    • 进入CloudFoundry选择已有应用
    • Environment Varieties
    • 选择USER-DEFINED，具体参数参照Parse使用文档
    • APP_ID xxxxxxx
    • MASTER_KEY xxxxxxx
    • PARSE_MOUNT /parse-server
    • REST_KEY xxxxxxx
