golang的restful框架选择

0.必备功能

坑多了，自然想多要一些糖，这样一个理想的RESTful框架，至少需要以下功能

  • RAML支持
  • CORS支持
  • JSON支持
  • 良好的Router机制
  • 简单的Controller、Service分离

  • ORM支持（最好带migrate）

    0.1 中间件大搜集

    RestGate

    支持框架：Negroni,Martini,Gin 为REST API提供endpoints的安全验证机制
