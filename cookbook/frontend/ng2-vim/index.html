用于angular2项目开发的vim环境配置及插件

可用于ng2开发的IDE评比

intellij idea ce 2016.1.1

  • 语法编辑 差
    • html/css emmet支持，默认补全<>
    • es6 完全不支持ES6，安装es6插件无效
    • ng2 完全不支持angular2
  • 项目特性 差
    • Rafactor 无法识别任何关联文件、命名、变量
  • 辅助工具 差

WebStorm 2016.2.1

  • 语法编辑 较好
    • html/css emmet支持，默认补全<>
  • 项目特性 较好
    • Rafactor 无法识别任何关联文件、命名、变量
  • 辅助工具 较好
    • gulp 小工具
    • RESTful Client 小工具
