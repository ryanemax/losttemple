ionicCreator实际体验与评测

ionicCreator实际体验与评测

免费版

  • 目前只支持ionic 1.0
  • 页面修改Routing URL，导出项目templates中page名未变化，依然是page1-n
  • components 组件少，且不可定制
  • 全程需要翻墙
  • 不实用，不如参考css文档，直接使用html写所需的页面

使用creator构建的一个测试项目：

https://creator.ionic.io/share/a5d70d07f53f

## 代理从creator创建项目到本地
http_proxy=http://127.0.0.1:8118 ionic start coinnova creator:a5d70d07f53f

## 登录ionic
cd coinnova
ionic login

## 修改后提交项目
ionic push

Pro版

  • 必须认证信用卡后，才可免费体验一个月
  • creator使用功能无明显提升
  • 允许1个私有项目

Team版

  • 必须认证信用卡后，才可免费体验一个月
  • creator使用功能无明显提升
  • 允许2个私有项目
