通胀与通缩并存的时代，如何投资最赚钱

  • 主办：金汇财经
  • 协办：MDF
  • 独家合作社交平台：脉脉
  • 主讲嘉宾：田洪良
    • www.tianhongliang.com
    • 微信：hjtz1963
    • QQ：595260321
    • 手机：13084150935

各投资品种比较

  1. 银行存款、国债
    • 万元，年亏损300元，人民币通胀率5-6%
  2. 股票、期货
    • 2017后小幅上升
  3. P2P（推荐）
    • 年收益 15% 存在欺骗、非法集资等风险
    • 年收益 10-13% 可信 => 认准平台 => 赶超通涨的收益率
  4. 贵金属、原油
    • 国内 => 无规范 => 无公信力 强烈不推荐
    • 国外 => 认准平台 => 新西兰、英国的监管标准完善
  5. 外汇（推荐）
    • 中国
    • 1994年，灰色地带，无限制无监管
    • 2005年，国外公司大量涌入
    • 2008、2009之后 更多人接触到
    • 2016年未完全开发/行业起步较晚、处于发展阶段
    • 国内金融的最后一块蛋糕
      • 分析师 100万/年收入
      • 交易员 300-500万/年收入
      • IB代理
      • 业务员 200-300万/年收入

外汇走势分析（2016-10）

  • 人民币美元
    • 美元预期走强（大选后加息）
    • 实际冲高回落
    • 高位震荡
  • 英镑美元(GBPUSD)
    • 大幅下跌，跌破最低点
    • 推荐：逢高做空
  • 澳元美元(AUDUSD)
    • 推荐：0.75 逢低做多
  • 美元加元(USDCAD)
    • 推荐：看涨
  • 现货黄金(AU)
    • 推荐：逢低做多
  • 现货白银
  • 原油
