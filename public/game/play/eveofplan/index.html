简介

帝国辖域以外的空间是土匪和海盗们的乐土，也是位于浩瀚星空的另一端的，那些投身于救助事业者的家园。EVE姐妹会就是其中最著名的组织，而且她不仅仅提供援助救济。协会建立在信仰和科学的基础上，这对奇异的组合吸引着帝国各处的人们参与其中。

姐妹会的最高领导人哈娜•杜拉多祭司。

她称彻底探知EVE之门的奥秘是“真神的旨意”，为了世间万物，她们会把工作进行到底。但迄今为止姐妹会的研究工作并没有太大进展，而上百万与姐妹会有共同愿望的人们正热切地期待着日后他们的研究会有新突破。

刷声望的技巧

艾拉肯 有一个空间站里面有姐妹会的2个1级任务代理人，没错的 ，就刷这个最快了，2级3级任务根本不考虑。直接刷1级任务到够4级任务的声望。

姐妹会和G/M关系很大 会加这两个势力的声望。故事线是加的，加四族声望，但一般任务可能会接到降低某四族声望的任务，拒绝就行了

参考资料

甲抗，转向速度中等，先天速度中等，武器是混合炮（动热伤害），部分船擅长无人机，混合炮的近炮种疾速炮超近的射程和超暴力的伤害，充分体验了盖伦特不是你死就是我亡的气质。磁轨炮和无人机的伤害都在近年得到加强，潜力很大。

电子战：Arazu/Lachesis

适合新人的： Vexor,Brutix（盾阔），钢板小米或者盾扩小米，多米

电子战（感应阻抑）路线：Maulus -> Celestis -> 最终目标：Arazu、Lachesis、Keres

无人机+磁轨炮路线： Tristan/Algos -> Vexor -> 小米-Myrmidon -> 最终目标：Ishkur、Dominix、Ishtar

炮术(Blaster or Railgun）路线：用炮护卫 -> Thorax -> Brutix -> 最终目标：Enyo、Talos、Deimos、Megathron、Daredevil

G族的pvp在早期很多船依赖无人机，但这种东西缺陷明显，无法对抗高速目标，常错失良机；在低安会被星门炮打掉，逃跑时来不及回收无人机。 Vexor,小米-myrm,多米-dominix就是通常所说的无人机船，一般偏重防御、毁电。 后期的发展目标是Ishtar，不过你需要有相当的操作水平才能玩好它。 Brutix是个不错的船，DPS极端强劲，几乎达到战列水平，因此很拉仇恨，建议不要带点，保证高速的同时注重防御。

等到你能开Arazu或者Lachesis，你就出头了。这两个船是小队抓人的神器，有着特别加长的反跳距离，而且能扫描抓人，血量也不赖，特别是Lachesis，应该可以认为是最硬的侦察舰。不过注意Lachesis你最好有重导技能，这样才能发挥多余的１００＋dps。

G族的PVP战列为甲抗，但是局限严重，除多米外一般不太适合小队战 其余PVP船只大多为盾抗，速度不弱，火力凶猛。

小航母 Ｇ族：甲抗，有战斗机攻击力加成，可以使用旗舰遥修甲，遥修盾 点评：战斗机攻击力加成在会战鸡肋，不要指望，刷怪的话这个航母在航母中是最好的（不过航母刷怪的通常都会在某天被人轮Ｘ）；小队战很不错的货，什么都能修，虽然不能传电，但是小队战一般也没有人需要电。

大航母 Ｇ族，甲抗：无人机攻击力加成，最强的对旗舰DPS，现在会战这个船常占所有旗舰的一半以上…可见其威力…

PVE/任务/刷怪

盖伦特 甲抗，甲抗优势同A族。 主要武器为磁轨炮，部分船使用无人机作为主要武器。

磁轨优势：炮船的一般优势，远距离能杀小怪，造成伤害及时之类；射程中上，伤害中下，失准伤害高于激光；pg消耗低，技能要求不如A族苛刻 缺点：转速较低，对于近距离的高速船容易打偏；耗电，虽然不如激光，但是总归是耗电；伤害固定：动热混合伤害，对于古斯塔，天蛇怪伤害高，天使怪伤害一般，萨沙，血袭伤害弱。

无人机优势：无人机刷怪，在最近一次更新后，变得不再鸡肋。无人机技能成型快、伤害灵活，但热能的garde才能发挥最大伤害，虽然射程受限。无人机不怕怪的电子战，例如ECM和炮扰等，同时无人机可以自动攻击目标，在确定怪不会打无人的情况下会比较轻松。 无人机的转速也非常强悍。

缺点：射程近，无人机速度奇慢，怪会攻击无人机等缺陷。
