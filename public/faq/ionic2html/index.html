ionic2常见问题汇总之界面布局篇

当icon图标没有outline模式时，点击后会变成空白

原因分析

因为ionic2自带的font-icon库中，改图标没有默认的outline样式，导致了点击后，显示空白。

http://stackoverflow.com/questions/36757738/ionic-angular-components-core-scss-error/36765721?noredirect=1#comment61154821_36765721

解决方案

scss文件中定义的样式不生效，@import已加入app.core.scss中

原因分析

命令行报错，导致后续scss样式无法被加载

Error: Import directives may not be used within control directives or mixins. on line 34 of node_modules/ionic-angular/components.core.scss @import "fonts/ionicons";

尝试一（无效）

npm install gulp-sass

npm install node-sass

尝试二（成功）

## vim node_modules/ionic-angular/components.core.scss
//@if ($ionicons) {
  @import "fonts/ionicons";
//}
